# 🏅 Olympic Games Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive **Olympic Games Analysis Dashboard** built using Power BI.  
It provides insights into athlete participation, medal distribution, and country-wise performance across different Olympic events.

---

## 🎯 Objectives
- Analyze historical Olympic data
- Identify trends in medal distribution
- Understand athlete demographics (age, gender)
- Compare performance across countries and cities

---

## 📌 Key Features

### 🔹 KPI Metrics
- Total Medals: 40K+
- Gold Medals: 13K+
- Silver Medals: 13K+
- Bronze Medals: 13K+
- Total Athletes: 136K+

### 🔹 Interactive Visualizations
- 🌍 **Map Visualization** – Athlete participation by city
- 📊 **Bar Chart** – Medal count by country/team
- 🍩 **Donut Chart** – Sports distribution by city
- 🥧 **Pie Chart** – Gender-wise participation
- 📈 **Area Chart** – Age distribution by season (Summer vs Winter)

### 🔹 Filters / Slicers
- Medal Type (Gold, Silver, Bronze)
- City Selection
- Season (Summer/Winter)
